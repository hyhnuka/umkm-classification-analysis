# UMKM Classification Analysis

Proyek ini bertujuan untuk melakukan analisis dan klasifikasi terhadap data Usaha Mikro, Kecil, dan Menengah (UMKM). Analisis ini mencakup pengolahan data untuk memahami karakteristik UMKM serta membangun model klasifikasi untuk menentukan status atau kategori UMKM berdasarkan dataset yang tersedia.

## Isi Repositori
- `umkm.ipynb`: Alur analisis data, visualisasi, dan pemodelan klasifikasi.
- `statistik.csv`: Data statistik dari dataset UMKM.
- `umkm_success.csv`: Dataset UMKM.
- `Makalah_DSC_Pertama-Kali-Ikut-Lomba.pdf`: Dokumentasi hasil proyek.

## Tujuan Analisis
- Melakukan eksplorasi data (EDA) untuk menemukan pola pada dataset UMKM.
- Memproses data untuk persiapan pemodelan.
- Membangun model machine learning untuk klasifikasi UMKM.
- Mengevaluasi performa model yang telah dibangun.
- Menghasilkan insight dari hasil anlisis.

## Kesimpulan Analisis
Berdasarkan hasil analisis, berikut adalah poin-poin kesimpulan utama:

1. **Faktor Penentu Keberhasilan:** Keberhasilan UMKM sangat bergantung pada pengelolaan manajerial dan pengalaman pemilik, terutama pada kualitas pencatatan keuangan, perencanaan bisnis, dan upaya pemasaran.
2. **Performa Pemodelan:** Model *Logistic Regression* (dengan penanganan ketidakseimbangan kelas) menunjukkan performa terbaik dengan akurasi mencapai 96%.
3. **Temuan Strategis:** UMKM yang memiliki catatan keuangan rapi, perencanaan bisnis terstruktur, dan mendapatkan pendampingan profesional memiliki peluang lebih besar untuk berhasil.
4. **Rekomendasi:** UMKM disarankan untuk mendigitalisasi pencatatan keuangan dan memanfaatkan pemasaran digital untuk meningkatkan jangkauan pasar. Pihak terkait dapat menggunakan model ini untuk memberikan pendampingan yang lebih tepat sasaran. Selain itu pemerintah dapat memberikan program kerja berkaitan dengan pengeloalaan awal bisnis, literasi keuangan dan digitalisasi kepada pelaku UMKM.
